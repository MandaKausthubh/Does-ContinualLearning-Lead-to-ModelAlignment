# Does LLM continual learning lead to model alignment?

## Introduction
I want you to implemente a project for if LLM continual learning can lead to model alignment.
Large language models exhibit biases related to gender, race, and religion due to patterns in pretraining data. While supervised fine-tuning (SFT) can reduce harmful outputs, it is off-policy and often causes catastrophic forgetting. Recent work on Self-Distillation Fine-Tuning (SDFT) shows that on-policy self-distillation can enable continual learning while preserving prior capabilities. This project investigates whether continual learning via self-distillation can serve as a stable mechanism for debiasing and model alignment.  

**Proposed Project/Experiments**: We will fine-tune a generative model such as LLama-1B/3B/8B on datasets generic datasets such as Dolly-15k, OASST1 while measuring the bias in these models (refer to the next section about this). This is proceeded by continual fine-tuning/fine-tuning on a gender alignment dataset such as StereoSet, while monitoring the bias of the previous datasets. 
 
**Measuring Bias**:
We have two prompt variants: The orignal prompt (input) and a flipped variant where we flip gendered words such as 'He/She' -> 'She/He', similarly we flip common male/female names into female/male names (random choice). For each prompt variant, the LLM will generate an output response. Two external evaluators will be applied: 

Sentiment Model S(.): Produces a continuous sentiment score. (RoBERTa-base-sentiment) 

Toxicity Model T(.): Produces a toxicity probability score. (Detoxify library/RoBERTA-hate-speech-dynabench-r4) 

Let o_i^(m) and o_i^(f) denote model outputs for male and female variants of sample i, which will be generated by subsampling the datasets and modification of the sentences with randomly changed pronouns, names, etc. The bias estimates are defined as: 

Sentiment Bias: 
B_sent = (1/N) * Σ_i [ S(o_i^(m)) − S(o_i^(f))]

Similarly, Toxicity Bias: 
B_tox = (1/N) * Σ_i [ T(o_i^(m)) − T(o_i^(f))]

Paired statistical tests will be used to assess whether mean differences significantly deviate from zero. Confidence intervals and effect sizes will be reported to quantify practical impact. 

Additional Stability Metric 
A decision flip rate will be measured to quantify how often gender substitution changes the model’s predicted label or reasoning outcome: 

F = (1/N) * Σ_i 1[ŷ_i^(m) ≠ ŷ_i^(f)] 

This captures the sensitivity of model behavior to demographic variation.
